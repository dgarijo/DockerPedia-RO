Supplementary materials for paper: DockerPedia: A Knowledge Graph of Software Images and their Metadata

This page provides pointers to the materials (vocabularies, dataset, software and queries) used in the paper DockerPedia: A Knowledge Graph of Software Images and Their Metadata.

Summary (extracted from the paper): This paper proposes a framework for automatically describing software images in a machine-readable manner by i) creating a vocabulary to describe software images; ii) developing an annotation framework designed to automatically document the underlying environment of software images and iii) creating DockerPedia, a Knowledge Graph with over 150,000 annotated software images, automatically described using our framework. We illustrate the usefulness of our approach in finding images with required software dependencies, comparing similar software images, addressing versioning problems when running computational experiments and flagging problems with vulnerable software dependencies.

Permanent URL of this page: https://w3id.org/dgarijo/ro/dockerpedia

Datasets.

There are two main datasets produced as a result of our publication:
  • The Dockerpedia Knowledge Graph dump: Turtle dump with the 150,000 annotated images from DockerHub. Images are annotated according to the DockerPedia ontology (see below for more pointers and information)
  • Query results used in the listing examples shown in the paper. Also available in GitHub.

Software

The DockerPedia framework has the following components:
  • DockerPedia annotator: Tool built on top of Clair for analyzing and annotating the contents of Docker images.
  • Vulnerability visualization.
  • DockerPedia website code. Includes examples and queries for the SPARQL endpoint.
  • SPARQL endpoint (note, only some examples are loaded, as the full dataset is over 20GB)
  • Sample images used to assess the quality of DockerPedia.

Ontology

The DockerPedia Ontology can be found in the following URL.
  • The latest version of the ontology is avilable at http://dockerpedia.inf.utfsm.cl/vocab#
  • The version 0.1.0 of the ontology documentation can be accessed at https://cbuil.github.io/dockerpedia-ontology/release/0.1.0/index-en.html
  • The version 0.1.0 of the TTL serialization of the ontology is avilable at https://cbuil.github.io/dockerpedia-ontology/release/0.1.0/ontology.ttl
